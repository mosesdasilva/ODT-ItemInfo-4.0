# Repository guidance

## Agent skills

### Issue tracker

Issues and specifications are tracked in GitHub Issues; external pull requests are not a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the standard five-role triage vocabulary. See `docs/agents/triage-labels.md`.

### Domain docs

This is a single-context repository. Read `CONTEXT.md` and relevant records under `docs/adr/` before changing behavior. See `docs/agents/domain.md`.
